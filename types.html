float A float in json is a number that has a decimal point. For example: 1.23

Notice the lack of quotes.

In Minecraft, some types MUST be typed as a float, even if you are using a whole number. In this case, include the 0, for example 1.0
integer An integer in json is a number that has no decimal point. For example: 1

Notice the lack of quotes.
boolean A boolean in json is a true or false value. For example: true

Notice the lack of quotes.

Booleans in Minecraft can often also be represented as 1 or 0 as well, using the integer or float types.
string A string in json is made up of a series of symbols (usually characters) that are grouped together inside of quotes.

For example "this is a string"

When you want to use a " character inside of a string, it must be escaped. This means that you use a \ character directly proceeding the quote. For example: "Hello there \"Friend\""
array An array in json is a list of values. For example: [1, 2, 3]

Notice how the values are separated by commas, and contained within square brackets.

All the rules that you already learned apply. For example, a list of strings would be represented as: ["string", "string", "string"]

Because arrays are a compound type, they can be combined infinitely. For example, an array, of arrays, of arrays...
object An object in json is a list of key-value pairs. For example: {"key": "value"}

Notice how the key-value pairs are separated by commas, and contained within curly brackets.

Here is an object containing a few different types:
{
"span" : "And Eggs!"
"amazing" : true
}
